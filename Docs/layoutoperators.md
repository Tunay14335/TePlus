** Layout System Components **

**>]** : Internal Command Executor

**o]** : Current Layout Indent
**+]** : Increase Layout Indent
**-]** : Decrease Layout Indent
**~]** : Mid Term Layout Indent (Decrease then Increase Layout)
**-+]** : Mid Term Layout Indent (Decrease then Increase Layout)


**n]** : Layout Indent Value , n : [0,...)

**#]** : Preprocessor
**$]** : Documentation Provider

**:]** : Line comment
**:+]** : Multi Line Comment (Begin)
**:-]** : Multi Line Comment (End)

***]** : Ignored Line

**^o]** : Sign Line Indent
**^-]** : Resign Line Indent
